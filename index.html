Item #: SCP-001

Object Class: Safe

Special Containment Procedures: SCP-001 is to be contained inside an incubator in a standard humanoid containment chamber. The incubator should be set to a constant 49 degrees celsius to be consistent with the egg's constant temperature. Any changes in temperature, whether from the egg or for the incubator, should be reported to the lead researcher immediately.

Description: SCP-001 is a pure black egg measuring 0.8x0.8x1.1 meters standing up. SCP-001 emanates a constant temperature of 49 degrees celsius. All attempts to view the contents of the egg show that it is empty.
